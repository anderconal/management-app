# management-app
### Getting started

1. Clone or download the GitHub repository:
  * https://github.com/anderconal/management-app.git
2. Go to the management-app directory:
  * cd management-app
3. Install node.js dependencies
  * npm install
  * This will automatically run bower install (which will install needed bower components)
4. Launch the management-app locally
  * From project's root directory:
  * grunt-serve
